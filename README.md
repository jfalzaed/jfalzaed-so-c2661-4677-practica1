# Ejecutor de Lotes — ST0257

Práctica del curso **Sistemas Operativos (ST0257)**.

Este repositorio contiene el desarrollo del _Ejecutor de Lotes_, un sistema
que simula la ejecución por lotes de un mainframe usando microservicios
que se comunican mediante tuberías nombradas.
